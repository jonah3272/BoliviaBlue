## Summary

- **Hub:** `/` = broad “dólar blue Bolivia” and main dashboard; not “just today” or “just live.”
- **Today:** `/dolar-blue-hoy` = “hoy” (calendar day, current quote).
- **Live:** `/dolar-paralelo-bolivia-en-vivo` = “en vivo” (real-time feel).
- **Question:** `/cuanto-esta-dolar-bolivia` = “¿cuánto está?” (quick number + conversions).
- **Brand/EN:** `/bolivian-blue` = “Bolivian Blue” (ES) and main **English** landing.
- **Educational:** `/que-es-dolar-blue` = what/why/how; no “today” angle.
- **Data:** `/datos-historicos` = past data and trends.
- **Tool:** `/calculadora` = convert; **Platform:** `/binance-p2p-bolivia`, `/usdt-bolivia`; **City:** La Paz, Santa Cruz, Cochabamba with same national rate + local framing.

Differentiation is achieved by **distinct H1s, first paragraphs, and supporting copy** plus **internal links** that reinforce each page’s role (no deletion or redirects).
